\doxysubsection{Detailed Description}
Wraps a macro expansion cursor and provides a common interface for a normal macro expansion cursor or a \char`\"{}pseudo\char`\"{} one.

\char`\"{}\+Pseudo\char`\"{} macro expansion cursors (essentially a macro definition along with a source location) are created in special cases, for example they can be created for identifiers inside macro definitions, if these identifiers are macro names.
