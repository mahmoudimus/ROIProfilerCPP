\doxysubsection{Detailed Description}
Obtain the offload kind, real machine triple, and an optional GPUArch out of the target information specified by the user. Bundle Entry ID (or, Offload Target String) has following components\+:
\begin{DoxyItemize}
\item Offload Kind -\/ Host, Open\+MP, or HIP
\item Triple -\/ Standard \mbox{\hyperlink{namespace_l_l_v_m}{LLVM}} Triple
\item GPUArch (Optional) -\/ Processor name, like gfx906 or sm\+\_\+30 In presence of Proc, the Triple should contain separator \char`\"{}-\/\char`\"{} for all standard four components, even if they are empty. 
\end{DoxyItemize}
