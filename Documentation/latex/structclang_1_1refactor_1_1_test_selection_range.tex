\doxysubsection{Detailed Description}
A source selection range that\textquotesingle{}s specified in a test file using an inline command in the comment. These commands can take the following forms\+:


\begin{DoxyItemize}
\item /$\ast$range=$\ast$/ will create an empty selection range in the default group right after the comment.
\item /$\ast$range a=$\ast$/ will create an empty selection range in the \textquotesingle{}a\textquotesingle{} group right after the comment.
\item /$\ast$range = +1$\ast$/ will create an empty selection range at a location that\textquotesingle{}s right after the comment with one offset to the column.
\item /$\ast$range= -\/$>$ +2\+:3$\ast$/ will create a selection range that starts at the location right after the comment, and ends at column 3 of the 2nd line after the line of the starting location.
\end{DoxyItemize}

Clang-\/refactor will expected all ranges in one test group to produce identical results.
