This Tool can be used to transform C++ Code. The Tool adds performance counter to a given file and automatically prints the generated statistics. The input parameters are a .cpp or .cxx file. The output file can be specified with -\/o output.\+cpp. With the --stmt parameter a specific statement can be traversed. If no --stmt is specified the programs annotates the main function and return the performance times and statement ids for further investigation.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{Author}\label{md__r_e_a_d_m_e_autotoc_md1}
